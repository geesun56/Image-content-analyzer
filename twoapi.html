Image search engine

  • Type keywords in the text box and press search button to search for pictures
  • You can select the number of pictures you want to see
  • Select pictures you like to analyze
  • Check the radio button to configure which analysis you want to take
  • Image tag: Analyzes the picture content and gives a prediction of what this picture is about with possibilities
  • Face Detection: Detects if there is a human face in a picture and gives a prediction of people's age range, race, gender with possibilities
